# To do list
A web app designed to make a note of to-do items. The frontend is built with ReactJS, while a JSON server is used to create RESTful APIs. 
Website: https://arv-repo-to-do-list.netlify.app/

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
